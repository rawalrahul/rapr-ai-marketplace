# Legal Notes

This repository is an open marketplace of RAPR AI wrapper packages.

Most entries are convenience packages that collect public links, install commands, metadata, and agent instructions. They are intended to help users discover and use public tools more easily.

The marketplace does not claim ownership of upstream projects. It does not relicense upstream projects. Users may always install directly from the upstream public repository or official documentation.

## License Fields

- `license`: license for the RAPR marketplace wrapper package.
- `upstream_license`: known upstream license when relevant.
- `license_scope`: plain-language explanation of what the package license covers.

## Contribution Review

Contributors should avoid submitting packages that include upstream source code, binaries, model weights, datasets, or proprietary documentation unless redistribution is clearly permitted and required notices are included.
